This program allows for a user to make a stack linked list
  of any data type. It come with four functions to use. It
  comes with push, pop, peak, and size. Pop will remove the
  nod eon the end of the list. Push will put a new node at
  the end of the list. Peak will return the data that is in
  the node on the end of the list. Finally the size function
  returns the number of nodes in the list at the time of call.
